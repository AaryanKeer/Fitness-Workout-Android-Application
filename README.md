# Android Fitness App

A complete Android fitness application built using **Kotlin**, **RoomDB**, and **Android Studio**.  
The app allows users to follow guided workout sessions by watching animated or model-based workouts, calculate their BMI, and track workout history using local storage.

---

## Features
- **Workout Sessions** – Follow on-screen model workouts and complete guided exercise sessions.  
- **BMI Calculator** – Quickly calculate Body Mass Index based on user height and weight.  
- **Workout History** – Automatically saves and displays completed sessions using **RoomDB**.  
- **User-Friendly Interface** – Clean and modern UI designed for easy navigation and motivation.  
- **Offline Support** – Data stored locally, no internet required.

---

## Built With
- **Kotlin** – Core development language  
- **Android Studio** – IDE for Android app development  
- **RoomDB** – For local storage of completed workouts and BMI data  

---

## How It Works
1. Select a workout session from the home screen.  
2. Follow the on-screen workout model to perform each exercise.  
3. After completion, your session is automatically saved to history.  
4. Use the **BMI Calculator** to check your fitness level.  
5. View past workouts and progress anytime from the **History** section.
